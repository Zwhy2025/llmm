# AI API 管理中心

一个现代化的前端项目，用于集中管理多个大模型API，支持OpenAI兼容模式，并提供API测试调用功能。

## 🚀 主要功能

### 1. API 管理面板
- **多API支持**：支持管理多个不同的AI API
- **OpenAI兼容**：完全兼容OpenAI API规范
- **类型分类**：支持OpenAI、Anthropic Claude、自定义等类型
- **状态监控**：实时显示API连接状态
- **CRUD操作**：完整的增删改查功能

### 2. API 测试控制台
- **实时测试**：直接在界面上测试API调用
- **参数配置**：支持模型选择、温度调节等参数
- **响应展示**：格式化显示API响应结果
- **使用统计**：显示token使用情况

### 3. 多设备数据同步
- **本地存储**：使用localStorage进行本地数据存储
- **自动同步**：可配置的自动同步间隔
- **数据备份**：支持数据导出和导入功能
- **云端扩展**：预留云端同步接口

### 4. 现代化界面设计
- **响应式设计**：支持桌面端和移动端
- **美观界面**：现代化的UI设计，支持深色/浅色主题
- **流畅动画**：丰富的交互动画效果
- **用户友好**：直观的操作界面

## 🛠️ 技术特性

### 前端技术栈
- **HTML5**：语义化标签，现代化结构
- **CSS3**：Flexbox/Grid布局，CSS变量，动画效果
- **JavaScript ES6+**：模块化编程，异步处理
- **Font Awesome**：图标库支持

### 核心功能
- **OpenAI兼容**：完全遵循OpenAI API规范
- **错误处理**：完善的错误处理和用户提示
- **数据持久化**：本地存储和同步机制
- **键盘快捷键**：提升操作效率

## 📦 安装和使用

### 1. 快速开始
```bash
# 克隆项目
git clone <repository-url>
cd ai-api-manager

# 直接打开index.html即可使用
# 或者使用本地服务器
python -m http.server 8000
# 或
npx serve .
```

### 2. 添加API
1. 点击"添加 API"按钮
2. 填写API信息：
   - **名称**：API的显示名称
   - **URL**：API端点地址（如：https://api.openai.com/v1）
   - **API Key**：认证密钥
   - **类型**：选择API类型
   - **描述**：可选的描述信息

### 3. 测试API
1. 切换到"API 测试"页面
2. 选择要测试的API
3. 配置测试参数（模型、温度等）
4. 输入测试消息
5. 点击"发送请求"查看结果

## 🔧 配置说明

### API类型支持
- **OpenAI兼容**：标准OpenAI API格式
- **Anthropic Claude**：Claude API格式
- **自定义**：自定义API格式

### 设置选项
- **数据同步**：启用/禁用自动同步
- **同步间隔**：设置同步频率（10-300秒）
- **主题选择**：浅色/深色主题切换

## 📊 数据管理

### 本地存储
- API配置存储在`localStorage`中
- 设置信息自动保存
- 支持数据导出/导入

### 数据格式
```json
{
  "apis": [
    {
      "id": "unique-id",
      "name": "API名称",
      "url": "https://api.example.com/v1",
      "key": "api-key",
      "type": "openai",
      "description": "描述",
      "createdAt": "2024-01-01T00:00:00.000Z",
      "status": "active"
    }
  ],
  "settings": {
    "syncEnabled": true,
    "syncInterval": 30,
    "theme": "light"
  }
}
```
